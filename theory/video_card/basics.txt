//Виды
- дискретные
- интегральные встроенные в процессор
- внешние

//Также
- Эталонные/референсные: amd, nvidia
- Неэталонные: создает вендор зачастую для флагманских решений

//Чип
- Занимает центральное место на плате
- состоит из блоков

//Основные блоки
- текстурные: натягивают текстуру на поверхность 3х-мерного объекта
- вычислительные ядра (ALU): занимаются в наложением света,  участвуют в рендере и конвертации видео
- блоки растеризации: формируют готовый кадр для отображения на монитор
- у блоков есть свои кэши и контроллеры

//Память
- vram
- hbm - находится на подложке gpu - ширина шины до 4096 бит
- gddr - соединена с gpu шиной - ширина шины до 384 бит у gddr6
- данные для обработки gpu получает из процессора и оперативной памяти
- чтобы хранить данные и записывать результаты используется видеопамять gddr 5-6

//Шина памяти
- канал м/у gpu и vram
- чем шире шина (больше разрядность) м/у gpu и vram тем больше gpu может обработать данных

//Пропускная способность памяти
- предельный объем обрабатываемой информации
- измеряется в гигабайтах в секунду

//Pci-express
- pci-express x16 содержит 16 линий для приема/передачи
- всего в pci х16 188 контактов
- 128 контактов для обмена данных
- 9 для питания видеокарты
- видеокарта может работать через любое количество линий
- например майнинг фермы подключаются через pcie x1, на в фермах обмен данными минимальный
- разъем питания может выдать до 75 ват
- карта может потреблять до 600вт
- для доп питания на карте есть доп разъемы: 6 пин (75вт), 8 пин (150вт) и 16 пин (600вт)

//1 линия pci-экспресс состоит из
- 2 контакта на передачу
- 2 контакта на прием
- 4 контакта заземления

//Рендеринг
- ЦПУ строит 3х-мерный объект по координатам вершин многоугольников
- текстурные блоки накладывают текстуры на объект
- на текстуре получаются артефакты
- фильтрация устраняет артефакты
- билинейная и трилинейная - простые алгоритмы - оставляют остаточные артефакты
- анизатропная устраняет все артефакты и устраняет размытие если текстура наклонена
- перед выводом на экран 3х-мерное изображение отображается 2х-мерным с помощью геометрических формул
- фильтрацию и вывод изображения выполняют блоки растеризации
- лесенку на границах кривых линий устраняет сглаживание
- с помощью затемнения пикселов создается рельеф текстуры, для этого нужны текстуры карты: карта нормалей, карта зеркальности и альбедо-карта
- теселляция увеличивает детализацию, добавляя дополнительные полигоны
- сглаживание, пиксельный шейдинг и теселляцию добавляют вычислительные ядра

//Количество узлов
Для отображения 1 сек fullhd видео 60fps нужно вычислить цвет для 1920х1080х60=124млн пикселов, поэтому
- текстурных блоков десятки
- блоков растеризации сотни
- вычислительных ядер тысячи

//Вычислительные ядра
- cuda - вычислительные ядра от nvidia
- rt-ядра - рассчитывают трассировку лучей
- тензорные ядра - проводят операции м/у огромными кластерами матрицами,
такие вычисления необходимы нейросетям,
а в частности dlss (nvidia) - увеличивает резкость кадрам с низким разрешением
