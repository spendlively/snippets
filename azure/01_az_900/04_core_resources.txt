//3. Storage services//
///////////////////////
 - Container (Blob) Storage
 - Disk Storage
 - File Storage
 - Storage Tiers

//Azure Storage Account (3 different types)
//1. General Purpose v2 gpv2 (main)
//2. General Purpose v1 gpv1 (legacy)
//3. Blob storage (legacy)

//General Purpose v2 gpv2 (main)
//can contain 4 types of files
 - Container (Blob) Storage
 - Tables
 - Queues
 - File Storage

1Gb worth 0.1$/month

//Options
 - access tiers - hot (default), cool (rare access to files - cheaper), archive (cheapest but slow)
 - performance tiers - standard or premium
 - location - different regions have different prices
 - redundancy / replication - allows to have a backup in another region
 - failover options
    - setting up a failover allows you to get files in different region in case of go down

//Disk Storage
 - pay for reservation
