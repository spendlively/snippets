////////
//lsof//
////////

lsof -u cindy | wc -l
выводит список файлов, открытых конкретным пользователем

lsof -U
все файлы сокетов домена Unix

lsof -cpython
сведения о файлах, которые держат открытыми процессы (python)

lsof +d /usr/bin
какие папки и файлы открыты в некоей директории (но не в её поддиректориях)

lsof -d 2
lsof -d 0-2
Список всех файлов ассоциированных с указанным файловым дескриптором.

lsof -p 1
lsof -p ^1
lsof -p 450,980,333
файлы, открытые процессом с PID

lsof -i
lsof -i tcp
lsof -i :25
Список всех сетевых соединений

lsof -t /var/log/dummy_svc.log
подавляет вывод всей информации за исключением ID процессов
