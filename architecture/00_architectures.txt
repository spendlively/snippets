
//Виды архитектур
1. Монолитная архитектура
 - Примеры
    - Clean Architecture Роберта Мартина
        - луковица из 4х слоев
    - Архитектура корпоративных приложений Мартина Фаулера
        - пирог из 3х слоев
 - Признаки
    - большой проект из нескольких независимых слоев
    - одна бд на все подпроекты
    - один домен
 - Дополнительно доменный слой может быть построен по принципам DDD

2. Микросервисная архитектура
 - проектирование строится вокруг api
