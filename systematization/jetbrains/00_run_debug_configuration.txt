
//Конфигурация для запуска приложения

//Используется для запуска приложения из IDE:
//phpstorm hot keys
[SHIFT] + [F10] - запуск текущей конфигурации (выбрать в выпвдвющем списке в верхнем правлом углу)
[CTRL] + [SHIFT] + [F10] - запуск с выбором конфигурации

//pycharm & goland hot keys
[SHIFT] + [F10] - запуск текущей конфигурации (выбрать в выпвдвющем списке в верхнем правлом углу)
[CTRL] + [SHIFT] + [F10] - запуск текущего файла
[ALT] + [SHIFT] + [F10] - запуск с выбором конфигурации

//Настройка конфигурации
Run -> Edit Configurations
В левом меню выбирается тип конфигурации или файл
В правом меню выбираются имя и опции конфигурации (интерпретатор, venv)
