    Модуль - файл .py, в котором содержится код на Python
        Переменная, класс, функция



//2 определения пакета
1. Установленный пакет
 - директория с файлами .py (модулями)
 - в python2 в папке пакета должен находиться файлик __init__.py (хотя бы пустой).
 - в python3 __init__.py необязателен, но желателен для совместимости.
2. Сторонняя библиотека
 - архив (zip|tar|egg|whl)
 - дополнение, приложение или утилита,
кот. можно устанавить из внешнего репозитория:
Bitbucket, Github, Google Code либо официального Python Package Index
