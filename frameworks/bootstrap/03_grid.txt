
//Разметка сетки
 - состоит из контейнера, рядов и колонок
 - содержимое должно быть расположено в колонках, и только колонки могут быть расположены в рядах
 - колонки сетки без установленного атрибута «ширина» автоматически получают равную ширину
 - колонки имеют горизонтальный паддинг padding для создания отступов между отдельными колонками,
но вы можете удалить марджины margin из рядов и паддинги padding из колонок, добавив класс .no-gutters в .row.
 - ширина отступа 30px (15px с каждой стороны столбца)



//Брейкпоинты
 - есть 5 брейкпоинтов: экстра маленький, маленький, средний, большой и экстрабольшой
 - каждый брейкпоинт сетки основан на минимальной широте,
т.е. они подходят для каждого вышестоящего яруса,
т.е. .col-sm-4подходит для маленьких, средних, больших и XL девайсов
 - col-{breakpoint}-auto - колонка, изменяющая свою ширину по ширине содержимого
 - row justify-content-md-center - выравнивание текста по центру для всего ряда



//Перенос на след строку
<div class="col">col</div>
<div class="col">col</div>
<div class="w-100"></div>
<div class="col">col</div>
<div class="col">col</div>



//Соответствие брейкпоинтов экранам устройств
Xs - экран смартфоеа вертикально
Sm - экран смартфона горизонтально
Md - экран планшета вертикально
Lg - экран планшета горизонтально или экран ноутбука
Xl - экран монитора



//Колонки
 - по умолчанию 12 колонок
 - если больше 12-ти колонок разместить в одном ряду,
каждая дополнительная колонка обернется в новую строку.



//Вложенность
 - Сетка может быть вложенной
 - .row->.col->.row->.col...


//Без пробелов между колонками




//Изменение порядка элементов
.col (2)
.col .order-12 (4)
.col .order-1 (3)
.col .order-first (1)




//Отступ
.col-md-3 .offset-md-3 - колонка размерам 3, после отступа в 3
.offset-md-0 - сброс отступа




//SASS
 - переменные и карты sass определяют число колонок,
ширину отступа и контрольную точку медиа-вызова,
на которой колонки начинают вести себя как плавающие
