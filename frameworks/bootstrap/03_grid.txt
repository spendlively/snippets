
//Ярусы
 - Есть 5 «ярусов» сеток, по одному для каждого «брейкпойнта»:
все контрольные точки (экстра маленький), маленький, средний, большой и экстрабольшой.
 - Ярусы сетки основаны на минимальной широте,
т.е. они подходят для каждого вышестоящего яруса
(т.е., .col-sm-4 подходит для маленьких, средних, больших и XL девайсов).
